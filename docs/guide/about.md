# About the author

**[Yeferson Segura](https://yefersonsegura.com/)** — mobile, web, and product-oriented software.

Omega **Architecture** exists in two sibling flavors:

- **Angular** — [`omega-angular` on npm](https://www.npmjs.com/package/omega-angular) and the **[Omega Angular docs](https://yefersonsegura.github.io/omega_angular/)**
