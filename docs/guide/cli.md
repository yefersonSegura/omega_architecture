# Omega CLI

The **`omega`** executable ships with the package (global activate or `dart run omega_architecture:omega …`).

Common commands:

| Command | Purpose |
|--------|---------|
| `omega create app <name>` | New Flutter app with Omega layout |
| `omega init` | Add Omega wiring to existing app (from app root) |
| `omega validate` / `omega doctor` | Health checks |
| `omega g ecosystem …` | Scaffold flow + agent + events |
| `omega ai …` | Optional Omi assistant (env, coach, explain traces) |

Full tables and flags: [COMANDOS_CLI.md](https://github.com/yefersonSegura/omega_architecture/blob/main/doc/COMANDOS_CLI.md).
