# Example app

The **`example/`** directory is a full Flutter application demonstrating:

- `OmegaScope`, initial flow activation, navigation  
- **Auth** module — `AuthFlow`, `AuthAgent`, login UI, intents and events  
- Debug-only **inspector** and optional **time-travel** (see repo README)

Run locally:

```bash
cd example
flutter run
```

Use it as the ground truth when aligning your own `omega_setup.dart` and module layout.
